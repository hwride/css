Image with srcset and pixel density descriptors

The pixel density descriptor of srcset lets you tell the browser what image should be used depending on the device pixel ratio. If there are 1x, 2x or 3x device pixels to CSS pixels then the corresponding image will be chosen.

You can see the device pixel ratio in the information in the top right of the screen under devicePixelRatio.

Note that the actual width of the device or window does not affect the calculation, only the device pixel ratio.
