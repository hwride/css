Percent height

Setting width and height with % values

Width and height can be set using % values. They are a percentage of the width/height of the element's containing block.

width: 400px; height: 100px;
width: 50%; height: 75%;

% height reverts to auto if the containing block has no set height

When a element has a % height and it's containing block has no set height then the element reverts to an auto height. This does not apply to width where % still takes an effect. Note the height and width of the final two blocks:

height: auto;
height: 200px;
height: 100%;
width: 50%;

Here is an example where the grandparent does has a fixed height. Note the grandchild is still using a height of auto.

height: 100px;
height: auto;
height: 100%;
