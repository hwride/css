Containing block

position: static, relative & sticky

For elements with a position of static, relative or sticky the containing block is formed by the edge of the content box of the nearest ancestor that is either a block container or establishes a formatting context. Note here:

  • block c has a width the size of the content box

a
width: 800px; padding: 30px;
b
width: 75%;
c
position: relative; top: 100%; left: 0; width: 100%;

position: absolute

For elements with a position of absolute, the containing block is formed by the edge of the padding box of the nearest ancestor that has a position other than static. Note here:

  • even though c is a child of b, it's Containing block is formed by a
  • block c has a width the size of the padding box
  • block c is positioned to the edge of the padding box when we set left: 0
a
position: relative; width: 800px; padding: 30px;
b
width: 600px;
c
position: absolute; top: 100%; left: 0; width: 100%;
